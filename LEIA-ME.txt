📋 INSTALAÇÃO RÁPIDA (3 PASSOS)
═══════════════════════════════════════════════════════════

1️⃣  INSTALAR PYTHON (se ainda não tiver)
   • Baixe em: https://www.python.org/downloads/
   • ⚠️  IMPORTANTE: Marque "Add Python to PATH"

2️⃣  EXECUTAR INSTALADOR
   • Clique duas vezes em: INSTALAR.bat
   • Aguarde a instalação (2-5 minutos)

3️⃣  CONFIGURAR API KEY (gratuita)
   • Obtenha em: https://aistudio.google.com/apikey
   • Clique em: CONFIGURAR.bat
   • Cole sua chave no arquivo .env


▶️  COMO USAR
═══════════════════════════════════════════════════════════

1. Clique duas vezes em: EXECUTAR.bat
2. O navegador abrirá automaticamente
3. Digite comandos em linguagem natural
4. Pronto!
